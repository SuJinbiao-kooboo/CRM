# CRM系统快速启动指南

## 环境要求

### 开发环境
- **JDK**: 1.8+
- **Node.js**: 12.0+
- **MySQL**: 8.0+
- **Maven**: 3.6+
- **IDE**: IntelliJ IDEA 或 Eclipse

### 数据库配置
1. 创建数据库：`CREATE DATABASE crm_system DEFAULT CHARACTER SET utf8mb4;`
2. 执行SQL文件：`database/crm_database.sql`
3. 执行初始化数据：`database/init_data.sql`

## 快速启动步骤

### 1. 后端启动
```bash
cd Back-Server
mvn clean install
mvn spring-boot:run
```
访问地址：http://localhost:8081

### 2. 前端启动
```bash
cd Front-Server
npm install
npm run dev
```
访问地址：http://localhost:8080

### 3. 默认登录
- 用户名：admin
- 密码：123456

## 主要功能模块

1. **系统管理** - 字典管理、用户管理
2. **线索管理** - 线索录入、跟进、转客户
3. **客户管理** - 客户信息、联系人管理
4. **订单管理** - 订单处理、状态跟踪
5. **库存管理** - 库存查询、导入导出
