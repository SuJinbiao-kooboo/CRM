# IT分销公司CRM系统 - 项目完成总结

## 项目概述

本项目是一个专为IT分销公司设计的完整CRM（客户关系管理）系统，采用前后端分离架构，实现了从线索管理到订单完成的全业务流程管理。系统针对IT分销公司"无库存、撮合交易"的业务特点进行了专门设计和优化。

## 技术架构

### 后端技术栈
- **Java 8** + **Spring Boot 2.x** - 核心框架
- **MyBatis-Plus** - ORM框架，提供强大的CRUD功能
- **MySQL 8.0** - 关系型数据库
- **Maven** - 项目管理和构建工具
- **Lombok** - 简化Java代码编写

### 前端技术栈
- **Vue 2.x** - 渐进式JavaScript框架
- **Element UI** - 企业级UI组件库
- **Vuex** - 状态管理模式
- **Vue Router** - 前端路由管理
- **Axios** - HTTP客户端
- **SCSS** - CSS预处理器

## 部署和使用

### 快速启动
1. **数据库初始化**: 执行database目录下的SQL文件
2. **后端启动**: 运行`mvn spring-boot:run`或使用IDE启动
3. **前端启动**: 执行`npm install && npm run dev`
4. **一键启动**: 使用提供的`start.bat`脚本

### 系统访问
- **前端地址**: http://localhost:8080
- **后端API**: http://localhost:8081
- **默认账号**: admin / 123456
